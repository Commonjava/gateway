Gateway

About

Gateway is an API facade to hide micro services from end users. It receives HTTP requests from clients (e.g., mvn) and forward to different services.

Rules

There is a classifier to determine the target service via Url patterns and HTTP methods. Click HERE to see the proxy rules.
